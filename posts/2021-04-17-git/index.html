Git 명령어 모음

 ETC      2021-04-16     66 words      One minute
Contents

Remote 변경

git remote remove origin

git remote add origin {새주소}

서브 모듈이 있는 경우 git clone

  • 서브 모듈이 있는 프로젝트는 git clone하면 서브 모듈의 폴더는 비어있게 된다.
  • 다음 명령어를 통해 서브 모듈에 관한 로컬 설정을 한 후, 리모트 데이터를 가져온다.
git submodule init

git submodule update
  • 또는 처음에 프로젝트를 clone할 때 다음 옵션을 붙여 실행하면 서브 모듈까지 업데이트 할 수 있다.
git clone --recurse-submodules https://github.com/like-g-su/{}
