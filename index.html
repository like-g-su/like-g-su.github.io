Hugo 이것저것

2021-02-10    ETC
새로운 post 추가 hugo new post/test.md 개발 서버 시작하기 hugo server -D http://localhost:1313 브라우저에서 변경 사항을 즉시 확인할 수 있습니다. 이미지 경로 지정 Docs > Content Management > Static Files 기본적으로 이미지 등 static 파일은 static/ 디렉토리 아래에 위치합니다. static/ 아래 파일들은 사이트의 루트 경로로 제공됩니다. (예) static/img/test.png는 http://{base-url}/img/test.png로 접근할 수 있습니다. 설정 파일에 직접 static 파일 경로를 설정할 수도 있습니다. config.toml staticDir = ["static_test"] 테마 Themes 원하는 테마를 Git submodule로 추가하여 사용합니다.

컨테이너와 가상머신 비교, 도커를 해야 하는 이유

2021-02-09    Docker
컨테이너 vs 가상머신 가상머신(Virtual Machine) 하이퍼바이저를 이용하여 여러 개의 OS를 하나의 호스트에 설치하여 사용합니다. 하이퍼바이저는 VM이 할당될 수 있도록 리소스를 물리 머신에서 분리합니다. 각각의 게스트 OS는 독립된 공간과 자원을 할당받아 사용합니다. VMware 장점 OS가 다른 VM은 동일한 물리적 서버에서 실행할 수 있습니다. 호스트OS, 다른 VM으로부터 완벽하게 격리되어 강력한 보안 정책이 필요할 때 유용합니다. 단점 하이퍼바이저를 거치기 때문에 일반 호스트에 비해 성능이 떨어집니다. 라이브러리, 커널, 애플리케이션이 있으면 VM의 크기는 수 기가 바이트 일 수 있습니다.
